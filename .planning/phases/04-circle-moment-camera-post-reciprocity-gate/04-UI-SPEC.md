## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline label padding |
| sm | 8px | Inset padding within rows, icon-to-text gaps |

- Front camera inset size: 25% of rear viewfinder width (per CONTEXT.md decision)

**Source:** AuthView.swift uses `padding(.horizontal, 32)`, `spacing: 16`, `spacing: 12`, `frame(height: 50)`, `cornerRadius: 12`. These anchor the scale.

### 4. CircleDetailView ŌĆö Reciprocity Gate Card

**Context:** Within the circle's Moments section (a LazyVGrid or VStack of member Moment cards)

## Interaction Contracts

### Active Window Banner
- Updates in real-time via a SwiftUI `Timer.publish(every: 1, on: .main, in: .common)` countdown
- Banner appears/disappears reactively when `isWindowActive` changes (no animation needed ŌĆö it either exists or doesn't)
- Tapping banner dismisses any active sheet/navigation first if needed, then presents camera

### Camera Capture
- Shutter button: scale animation on press ŌĆö `scaleEffect(isPressed ? 0.92 : 1.0)` with `.spring(response: 0.2)`
- After capture: brief white flash overlay (`.white.opacity(flashOpacity)`) that fades from 1 to 0 over 0.3 seconds
- Device capability check on appear: if `AVCaptureMultiCamSession.isMultiCamSupported == false`, fall back to single rear camera silently (no UI change visible to user)

### Post Flow
- Posting is non-dismissible while in progress (no swipe-to-dismiss on sheet, button disabled)
- On success: dismiss PreviewView and CameraView together (dismiss both layers), CircleDetailView reloads Moments
- On success: all locked cards in CircleDetailView reload and unblur (re-fetch all today's Moments)
- On failure: `"Failed to post. Try again."` appears as a non-blocking banner below the Post button, button re-enables

### Reciprocity Gate Unlock
- After successful post, `CircleDetailView` fetches updated Moments list
- Cards transition from blurred-locked to unblurred-unlocked: `.animation(.easeOut(duration: 0.4), value: hasPosted)`
- No individual card animation delay (all unlock simultaneously)
