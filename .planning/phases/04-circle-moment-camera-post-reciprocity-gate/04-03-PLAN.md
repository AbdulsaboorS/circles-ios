    **1. Create `Circles/Moment/MomentCardView.swift`:**

    This view renders a single Moment card in three states: locked, unlocked, and own-unposted.

    ```swift
    import SwiftUI

    struct MomentCardView: View {
        let moment: CircleMoment?       // nil for own-unposted state
        let isOwnPost: Bool             // true if moment.userId == currentUser
        let hasPostedToday: Bool        // true if current user has posted in this circle today
        let onTapLocked: () -> Void     // opens camera
        let memberName: String          // display name (UUID prefix for now)
    }
    ```

    **Three states:**

    **A) Unlocked card** (`hasPostedToday == true && moment != nil && !isOwnPost`, OR `isOwnPost == true`):
    - `AsyncImage(url: URL(string: moment!.photoUrl))` filling card
    - `.aspectRatio(3/4, contentMode: .fill)` clipped
    - `cornerRadius(12)`
    - If `moment!.isOnTime`: top-right star badge -- 16pt white circle background, SF Symbol `star.fill` 12pt `Color(hex: "E8834B")`, 6pt inset from corner
    - Below card: member name in `.caption`, `.white.opacity(0.7)`, single line `.lineLimit(1)`

    **B) Locked card** (`hasPostedToday == false && moment != nil && !isOwnPost`):
    - Same `AsyncImage` but with `.blur(radius: 20)` applied to the image content
    - Overlay stack on top of blur:
      - `Color.black.opacity(0.4)` scrim
      - SF Symbol `lock.fill`, 24pt, white, centered
      - Text `"Post to unlock"`, `.caption`, `.white.opacity(0.85)`, 4pt below lock
    - `cornerRadius(12)`
    - `.onTapGesture { onTapLocked() }`
    - Below: member name in `.caption`, `.white.opacity(0.7)`
    - Accessibility label: `"\(memberName)'s Moment -- locked. Tap to post your Moment first."`

    **C) Own unposted** (`moment == nil && isOwnPost`):
    - `LinearGradient(colors: [Color(hex: "1A1D35"), Color(hex: "0D1021")], startPoint: .topLeading, endPoint: .bottomTrailing)`
    - `.aspectRatio(3/4, contentMode: .fill)`
    - Same blur overlay layout but on the gradient
    - `cornerRadius(12)`
    - `.onTapGesture { onTapLocked() }`
    - Below: `"You \u{2014} tap to post"`, `.caption`, `Color(hex: "E8834B")`

    Card width: let the LazyVGrid determine it (2-column adaptive grid in CircleDetailView).

            MomentPreviewView(
                image: image,

                    let _ = try await MomentService.shared.postMoment(
