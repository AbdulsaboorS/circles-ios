## ⚠ User Action Required Before Wave 2

Apply the migration SQL **before executing Plan 06.2-04** (Public Explore tab needs the column to exist):

```sql
ALTER TABLE circles ADD COLUMN IF NOT EXISTS is_public BOOLEAN DEFAULT false;
```

Paste this in **Supabase Dashboard → SQL Editor** and run it. To make a circle public for testing:

```sql
UPDATE circles SET is_public = true WHERE id = '<your-circle-uuid>';
```
