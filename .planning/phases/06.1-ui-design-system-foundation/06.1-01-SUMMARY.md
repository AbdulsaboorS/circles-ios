## What Was Built

### DesignTokens.swift

Primitive color tokens:
- `Color.darkBackground` = #0E0B08 (warm near-black, D-01)
- `Color.lightBackground` = #F5F0E8 (warm cream, D-02)
- `Color.accent` = #E8834B (amber, D-03)
- `Color.darkBlob` = #1A3A2A (forest green, D-11)
- `Color.lightBlob` = #EDE0C8 (warm beige, D-12)
- `Color.darkTextPrimary/Secondary`, `Color.lightTextPrimary/Secondary`, `Color.lightCardSurface`

`AppColors` semantic resolver: `AppColors.resolve(_ scheme: ColorScheme)` returns the full token set for the given color scheme.

D-07 static semantic alias extensions: `Color.appBackground`, `Color.cardSurface`, `Color.textPrimary`, `Color.textSecondary`, `Color.blobPrimary`, `Color.blobSecondary`

Font tokens:
- New York serif: `Font.appHeroTitle` (34pt), `Font.appTitle` (28pt), `Font.appHeadline` (22pt)
- SF Pro: `Font.appBody` (17pt), `Font.appSubheadline` (15pt), `Font.appCaption` (13pt), `Font.appCaptionMedium` (13pt medium)

### ThemeManager.swift

`ThemeMode` enum: `.auto`, `.alwaysLight`, `.alwaysDark` — persisted to UserDefaults key `"themeMode"`.

`ThemeManager.shared` — `@Observable @MainActor` singleton:
- `colorScheme: ColorScheme` — apply via `.preferredColorScheme(ThemeManager.shared.colorScheme)` at root
- `mode: ThemeMode` — changing mode triggers immediate apply + timer rescheduling
- `scheduleAutoSwitch()` — reads `cityLatitude`/`cityLongitude` from UserDefaults; falls back to 6am–8pm heuristic if no location
- `SolarCalculator` private enum: NOAA solar calculation algorithm, no external dependencies

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] Adhan SPM package not in project**
- **Found during:** Task 2 (ThemeManager)
- **Issue:** Plan stated "Adhan already in SPM" but Package.resolved has no Adhan entry. No `import Adhan` exists anywhere in the codebase. The Adhan reference was to Deno/TypeScript Edge Functions, not the iOS app.
- **Fix:** Replaced Adhan dependency with a self-contained `SolarCalculator` private enum using the NOAA solar calculation algorithm. Accurate to ±1 minute for latitudes ±60°. Same interface: returns `(sunrise: Date, sunset: Date)?` from lat/lng + date. No external packages needed.
- **Files modified:** Circles/DesignSystem/ThemeManager.swift
- **Commit:** b8c163b

**2. [Rule 1 - Bug] Swift 6 Sendable closure mutation error**
- **Found during:** Task 2 (build verification)
- **Issue:** `scheduleTimer` took `@Sendable` closure, but the `[weak self]` closures mutated `@MainActor`-isolated `colorScheme` directly — Swift 6 rejects this.
- **Fix:** Wrapped mutations in `Task { @MainActor [weak self] in ... }` inside each closure.
- **Files modified:** Circles/DesignSystem/ThemeManager.swift
- **Commit:** b8c163b (same commit)
