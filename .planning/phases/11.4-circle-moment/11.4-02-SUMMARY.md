# 11.4-02 Summary — Multi-Circle MomentService + Preview Disclaimer

## What Was Done

- `MomentService.uploadPhoto(image:userId:)` — uploads to `shared/{userId}_{date}.jpg` (no circleId in path); same URL reused across circles
- `MomentService.postMomentToAllCircles(image:circleIds:userId:caption:windowStart:) async throws -> MomentPostResult` — loops over circleIds, inserts one row per circle, collects successes and failures
- `MomentPostResult` struct: `succeeded: [CircleMoment]`, `failedCircleIds: [UUID]`, `isFullSuccess`, `isPartialSuccess`, `totalCount`
- `MomentError.noCircles` and `MomentError.allInsertsFailedCircles` added
- `MomentPreviewView` updated: new `circleCount: Int` parameter; shows "This will be shared to all your circles (N)" disclaimer when `circleCount > 1`
- `CommunityView`: passes `circleCount: viewModel.circles.count`
- `CircleDetailView`: passes `circleCount: 1` stub (fixed in Plan 04)

## Key Interfaces Produced

```swift
func postMomentToAllCircles(
    image: UIImage,
    circleIds: [UUID],
    userId: UUID,
    caption: String?,
    windowStart: String?
) async throws -> MomentPostResult

struct MomentPostResult {
    let succeeded: [CircleMoment]
    let failedCircleIds: [UUID]
    var isFullSuccess: Bool
    var isPartialSuccess: Bool
    var totalCount: Int
}
```
