# Phase 11.4 — Circle Moment (BeReal Mechanic)

**Goal:** Finalize the Circle Moment feature end-to-end — from prayer-time notification trigger through dual-camera capture, posting, and feed display. This is the core social mechanic of Circles.

---

## Product Decisions (locked)

### Trigger
- 30-minute window, randomized across the day's prayers (not always the same prayer)
- Push notification fires when the window opens
- Window: 30 minutes from notification send time

### Late Posting
- Users CAN post after the window closes — encouraged, not punished
- Late posts show a small emoji badge on the feed card (e.g. 🕰) — funny, not demeaning
- No shame language anywhere

### Camera
- Dual camera kept (front + back simultaneously) — already partially built
- Single capture (no retake tracking, no retake count display)

### Distribution
- For now: 1 moment goes to ALL circles the user is in
- Disclaimer shown on preview screen: "This will be shared to all your circles"
- Multi-circle targeting deferred to a future phase

### What NOT to build
- Retake counter / tracking
- Late post shame ("X hours late")
- Public/cross-circle feed moments
- BeReal-style "see others only after posting" gate (we have Reciprocity Gate already from Phase 8)
